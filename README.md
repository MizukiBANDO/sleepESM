# 睡眠記録アプリ　「ねむログ」
---
#### **📒✏️「ねむログ」とは？**
経験サンプリング法の手法を採用し，睡眠と睡眠に関連する行動の測定を可能にする記録アプリです。お手元のスマホやPCで簡単に記録を開始できます。
